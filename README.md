# Car Price Prediction

## Step 1

### Clone Repository
```bash
git clone https://github.com/akhil-s-kumar/Car-Price-prediction.git
```

## Step 2

### Change directory
```bash
cd Car-Price-prediction
```

## Step 3

### Install dependencies

```bash
pip install requirements.txt
```

## Step 4

### Run application

```
app.py
```

## Step 5

### Application will be running at

`http://127.0.0.1:5000`
